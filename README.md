Emmet-DocSet

[Emmet](http://emmet.io/) (formerly Zen Coding) DocSet for [Dash](http://kapeli.com).

`0.1.2`

### Update

	+ deprecated old build.py

### Requirements

* Python 2.7+
* Git
* Homebrew
* Phantomjs


### Usage

~~$ python build.py~~

	$ brew update & brew install phantomjs
	$ chmod +x build
	$ ./build
